About

Selected Projects

Research and engineering projects in surgical video understanding, medical vision, temporal modeling, 3D perception, and robotic systems.

Surgical Keypoint Localization

Mask-Guided Structural Priors for Surgical Tool Keypoint Localization

Built a multi-frame surgical video model for tool tip and anchor localization, integrating SAM2-derived mask priors with heatmap-based coarse-to-refine prediction. Implemented PyTorch training, inference, multi-GPU execution, heatmap generation, and quantitative evaluation.

Surgical Video Keypoint Detection SAM2 PyTorch

Retinal Motion Compensation

Predictive Retinal Motion Compensation for Robotic Subretinal Injection

Developed an iOCT and RGB-D vision pipeline for retinal motion compensation. Segmented retinal membranes, applied RANSAC filtering to remove outliers, extracted needle-to-retina distance signals, and benchmarked LSTM and Mamba models for temporal motion forecasting.

Medical Robotics RGB-D RANSAC Mamba

Monocular 3D Perception

Monocular 3D Human Pose and Metric Depth Estimation

Built a monocular 3D perception pipeline combining YOLOv8 keypoint detection and Depth Anything V2 dense depth estimation. Calibrated monocular depth scale using known human height and 2D bounding box height, then back-projected 2D keypoints into 3D using camera intrinsics.

3D Vision YOLOv8 Depth Anything V2 Camera Intrinsics

Temporal Surgical Tracking

Temporal Decoder for Surgical Tool Tracking

Developed a temporal decoder for surgical tool representation tracking. Extracted frame-level visual features with an HRNet backbone and applied transformer-based temporal modeling to improve video-level consistency for surgical action and tool state prediction.
